# Format the response of a crul::HttpClient from json to list

Format the response of a crul::HttpClient from json to list

## Usage

``` r
response_to_list(response = NULL)
```

## Arguments

- response:

  "HttpResponse" as returned by client\$get where client was produced by
  [`proj_idf_client()`](https://lsteinmann.github.io/idaifieldR/reference/proj_idf_client.md)

## Value

a list
