# Reduces a (relations) list to its first element and warns accordingly.

Reduces a (relations) list to its first element and warns accordingly.

## Usage

``` r
reduce_relations(relation, uuid = NA, identifier = NA)
```

## Arguments

- relation:

  The resources relation-list to be reduced

- uuid:

  The id/UUID of the resource (for warning)

- identifier:

  The identifier of the resource (for warning)

## Value

The reduced relations list
