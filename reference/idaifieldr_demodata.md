# idaifieldR Demodata

Test and demonstration data to illustrate the packages functionality,
has no real world representation.

## Usage

``` r
idaifieldr_demodata
```

## Format

### `idaifieldr_demodata`

A list (`idaifield_docs`) with 108 elements:

- identifier:

  identifier / Kurzbezeichner

- all other values:

  entries in the field database

## Source

Random collection of imagined data
